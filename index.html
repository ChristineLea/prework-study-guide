HTML

  • HTML files start with the !DOCTYPE declaration
  • Elements and content contained within the head element are displayed to the user
  • Content that you want a user to see and interact with must be palced inside the opening and closing body tags
  • HTML defines the structure or framework of a webpage
  • Semantic elements assist in SEO and accessibility

CSS

  • CSS adds style to an HTML webpage
  • CSS rules are declared using CSS selectors to select and style corresponding HTML elements
  • A margin indicates how much space is wanted around the outside of an element.
  • Padding indicates how much space is wanted around the content inside an element.
  • External CSS style sheets are added to an HTML document within the head element, using the link element.

Git

  • git status: Check which branch you are in
  • git checkout main: Switch to main branch
  • git pull origin main: Pull the remote branch in to the local main branch to sync remote/local main branch
  • git checkout -b feature/branch-name: Creates a new branch and moves you into the branch
  • touch file.extension: touch keyword creates a new file for example script.js
  • git add -A: Add the changed files in the local working directory to the staging area.
  • git commit -m "enter message": Commit changes to repository.
  • Make sure current branch is up to date before opening a pull request. Pull in the base branch.
  • git push origin feature/branch-name: Push up all of the changes.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • An if statement is used to check if a condition is true. If it evaluates to true, the code block will execute.
  • An else if statement allows for more than one condition to be checked.
  • A for loop has three conditional statements that it requires.
  • The for loops conditional statements are: starting point, the condition to check, and what to do everytime the code loops.
  • Arrays can store a group of data.
  • Individual array items can be accessed by referencing their array index number, which starts from zero.
  • A function is a set of reusable instructions.
  • Unlike an if statement or a for loop, functions do not automatically execute.
  • To execute a code block in the function declarations body, the function needs to be called, or invoked.
  • The syntax to call a function is: functionName();
  • Functions can be used to iterate over an array.
  • The built-in Math object can be used to generate a random number. The Math functions used to perform this are Math.random() and Math.floor().
